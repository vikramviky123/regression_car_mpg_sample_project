Model Analysis

1. Preprocessing

In the preprocessing phase, the data is transformed to prepare it for model training. Numeric variables are imputed with the mean, while categorical variables are imputed with the mode using SimpleImputer. The data is then scaled using StandardScaler. A pipeline is used to streamline both processes simultaneously.

2. Ensemble of 6 Models for Training

Training involves the use of six different models for regression:

  • Linear Regression
  • Decision Tree Regression
  • Random Forest Regression
  • Historical Gradient Boosting Machine Regression
  • XGBoost Regression
  • LightGBM Regression

3. Model Training

Model training is performed using k-fold cross-validation with six splits. The average of predicted values is computed to evaluate the model's performance.

4. Metrics Chosen

The following metrics are chosen to assess model performance:

  • R-squared (R2)
  • Mean Absolute Error (MAE)
  • Root Mean Squared Error (RMSE)
